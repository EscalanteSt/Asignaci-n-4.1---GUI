# Tarea 4.1 - GUI: Formulario Gráfico con Clase de Procesamiento

Este repositorio contiene la implementación de una interfaz gráfica de usuario (GUI) desarrollada en Java utilizando Apache NetBeans. El proyecto demuestra la correcta aplicación de los principios de encapsulamiento y separación de responsabilidades.

## 📋 Descripción del Proyecto
El programa permite capturar los datos de un estudiante (Matrícula, Nombres, Apellidos y Cuatrimestre) a través de un formulario visual (`JFrame`) y procesarlos utilizando una clase dedicada e independiente (`ProcesarDatos`), la cual maneja la información mediante atributos privados.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java
* **Entorno de Desarrollo (IDE):** Apache NetBeans
* **Librerías Visuales:** Java Swing (`JFrame`, `JLabel`, `JTextField`, `JButton`, `JOptionPane`)

## 🧱 Estructura Principal
El código fuente (`src/principal/`) está dividido en dos componentes clave:
1. `InterfazParticipante.java`: Maneja exclusivamente el diseño visual (Matisse) y la captura de eventos de los botones.
2. `ProcesarDatos.java`: Clase de procesamiento lógico que almacena y formatea los datos encapsulados.
